# Keep Clone Website - HTML, CSS, JS

A simple note-taking web application inspired by Google Keep, built with HTML, CSS, JavaScript, and Bootstrap.

## Features

- **User Authentication**: Sign up and sign in functionality with local storage
- **Create Notes**: Add notes with titles, content, and tags
- **Tag Management**: Create, edit, and delete custom tags
- **Search & Filter**: Search notes and filter by tags
- **User Profile**: Update account settings
- **Responsive Design**: Modern UI built with Bootstrap 5

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5.1.3
- LocalStorage for data persistence

## Pages

- `index.html` - Landing page
- `sign.html` - Sign in page
- `registration.html` - Registration page
- `home.html` - Main notes page
- `tags.html` - Tag management
- `user.html` - User profile settings

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Keep-Clone-Website.git
   ```

2. Open `index.html` in your browser to start

3. Create an account and start taking notes!

## Usage

1. **Sign Up**: Create an account with email and password
2. **Sign In**: Log in with your credentials
3. **Create Notes**: Add notes with tags for organization
4. **Manage Tags**: Create custom tags for categorizing notes
5. **Search**: Find notes quickly with the search feature
6. **Filter**: View notes by specific tags

## Data Storage

All data is stored locally in your browser using LocalStorage:
- User accounts
- Notes (per user)
- Tags (per user)

**Note**: Clearing browser data will delete all stored information.
